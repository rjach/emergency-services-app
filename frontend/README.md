# Frontend (Plain HTML/CSS/JS)

Minimal frontend scaffold for testing backend APIs.

## Files

- `index.html`: Main page.
- `style.css`: Basic styling.
- `script.js`: Example API call to backend health endpoint.

## Run Frontend Locally

You can open `index.html` directly in the browser or run a static server.

Example using Python:

```bash
cd frontend
python3 -m http.server 5500
```

Then open: <http://localhost:5500>
