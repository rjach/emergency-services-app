# Emergency Services App (Scaffold)

This repository is intentionally scaffold-only for a college assignment.

## Project Structure

- `frontend/`: Plain HTML, CSS, and JavaScript app.
- `backend/`: Node.js + Express + MongoDB starter API.
- `docs/`: Beginner-friendly documentation for setup and team workflow.

## Quick Start

1. Read `docs/guides/getting-started.md`.
2. Setup backend using `docs/development/backend-setup.md`.
3. Run frontend and test API calls using `docs/guides/frontend-api-testing.md`.

## Notes

- No business logic has been implemented yet.
- Teams should add models, routes, controllers, and frontend features as needed.
